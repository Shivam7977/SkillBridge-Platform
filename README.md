# SkillBridge 🚀

SkillBridge is an AI-powered, full-stack developer platform designed to centralize learning, project building, and career growth. It provides users with automated tools to generate professional portfolios, ATS-friendly resumes, and dynamic learning roadmaps.

## ✨ Features
* **🤖 AI Roadmap Generator:** Powered by the Google Gemini API, it creates custom, step-by-step learning paths based on user goals and fetches relevant YouTube tutorials.
* **📄 One-Click Resume Builder:** Automatically compiles user profile data and project history into a professionally formatted, downloadable A4 PDF.
* **🌐 Dynamic Portfolio Builder:** Generates standalone, stylized HTML portfolio websites using real-time data injection.
* **💬 AI Platform Assistant:** A built-in, floating Chatbot that guides users through the platform and answers questions using advanced natural language processing.
* **👥 Community Hub:** Users can upload projects, browse an ecosystem of other builders' work, and send direct messages.

## 🛠️ Tech Stack
* **Backend:** Python, Flask
* **Database:** MongoDB
* **Frontend:** HTML5, CSS3, JavaScript, GSAP (Animations)
* **AI Integration:** Google Gemini API (`gemini-pro` / `gemini-1.5-flash`)
* **Libraries/Tools:** Flask-Login, Flask-Bcrypt, html2pdf.js
