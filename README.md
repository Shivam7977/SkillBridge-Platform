## ⚙️ Installation & Setup

1. Clone the repository

2. Install dependencies:
   pip install -r requirements.txt

3. Create `.env` file and add:
   MISTRAL_API_KEY
   MONGO_URI
   MAIL_USERNAME
   MAIL_PASSWORD
   YOUTUBE_API_KEY

4. Run the app:
   python app.py
